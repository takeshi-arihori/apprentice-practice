# apprentice-practice
アプレンティスのカリキュラム用リポジトリです

### ルール
1. week別にフォルダーを作成し、それぞれのカリキュラムに沿って学習
2. 日々の日報はdaily-reportにアップする
3. 学習する前にはissueを立てる
